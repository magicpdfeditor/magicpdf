# PDF Magic - Free PDF Tools

A modern, feature-rich PDF manipulation website similar to "I Love PDF". Built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

- **Merge PDFs** - Combine multiple PDF files into one
- **Split PDFs** - Extract specific pages from a PDF
- **Compress PDFs** - Reduce file size while maintaining quality
- **Convert to PDF** - Convert images and documents to PDF
- **Extract Text** - Extract text from PDF files
- **Add Watermark** - Add text or image watermarks to PDFs
- **Word to PDF** - Convert Word documents to PDF
- **PowerPoint to PDF** - Convert presentations to PDF
- **Excel to PDF** - Convert spreadsheets to PDF
- **PDF to JPG** - Convert PDF pages to images
- **JPG to PDF** - Convert images to PDF
- **Rotate PDF** - Rotate pages by 90°, 180°, or 270°
- **HTML to PDF** - Convert HTML content to PDF
- **Edit PDF** - Edit text and content in PDF files
- **Sign PDF** - Add digital signatures to PDFs
- **Unlock PDF** - Remove password protection from PDFs
- **Protect PDF** - Add password protection to PDFs
- **Organize PDF** - Merge and organize multiple PDFs
- **Page Numbers** - Add page numbers to PDF documents
- **Redact PDF** - Remove sensitive information from PDFs
- **Crop PDF** - Crop margins from PDF pages
- **PDF to PDF/A** - Convert PDF to archival PDF/A format
- **Repair PDF** - Fix corrupted or damaged PDF files
- **Scan PDF** - Enhance and optimize scanned documents
- **OCR PDF** - Extract text from scanned PDFs using OCR
- **Compare PDF** - Compare two PDF files to find differences

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **PDF Processing**: pdf-lib, pdfjs-dist
- **Backend**: Next.js API Routes

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
